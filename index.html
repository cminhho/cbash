CBASH CLI

Command-line tools for developers (macOS/Linux). Automate tasks, manage git, run dev services, and use helpers for AWS, K8s, and docs.

Quick install (macOS):

brew tap cminhho/cbash https://github.com/cminhho/cbash
brew install cbash-cli

Or:

sh -c "$(curl -fsSL https://raw.githubusercontent.com/cminhho/cbash/master/tools/install.sh)"

What it does

  • Setup – Check dev environment, install tools (brew), create workspace
  • Git – Branches, squash, pull-all, backup, open in browser
  • Dev – Start/stop Docker services, logs, exec, stats
  • Docker – Aliases and helpers (dps, drm, prune, kill-all)
  • AWS – SSH via SSM, SQS (localstack), SSM params
  • Kubernetes – Pods, logs, exec, rollout restart, cheat
  • Generators – Troubleshooting/feature dirs, workspace, project, docs
  • Docs & Cheat – View/edit docs and cheatsheets
  • AI – Chat with Ollama, list/pull models
  • macOS – Lock, speedtest, ports, IPs, passgen, tree, find/replace
  • Proxy – Enable/disable proxy (env, npm, git)

Usage

cbash                  # Show help
cbash <plugin>         # Run plugin (e.g. git, dev, aws)
cbash list-plugins     # List available plugins
