CBASH CLI
GitHub Docs

One CLI for your daily dev workflow

Git, Docker, AWS, Kubernetes, generators, docs, and more. Install once, use everywhere on macOS and Linux.

Install (macOS)
brew tap cminhho/cbash https://github.com/cminhho/cbash
brew install cbash-cli

Other options: curl, wget, manual

What’s included

Setup

Check dev environment, install tools via Homebrew, create workspace structure.

Git

Branches, squash, pull-all repos, backup, open in browser, aliases.

Dev & Docker

Start/stop services, logs, exec, stats. Aliases: dps, drm, prune, kill-all.

AWS

SSH via SSM, SQS (localstack), SSM parameters.

Kubernetes

Pods, logs, exec, rollout restart, kubectl cheat sheet.

Generators

Troubleshooting/feature dirs, workspace, project, doc templates.

Docs & Cheat

View and edit docs and cheatsheets from the terminal.

AI (Ollama)

Chat, list and pull models from the CLI.

macOS & Proxy

Lock, speedtest, ports, IPs, passgen, tree. Enable/disable proxy.

Quick start

cbash                  # Show help
cbash <plugin>         # Run plugin (git, dev, aws, …)
cbash list-plugins      # List available plugins
